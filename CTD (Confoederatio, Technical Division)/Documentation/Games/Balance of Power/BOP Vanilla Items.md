**Vanilla Items** or Resource Gathering Operations (RGOs) are typically used as resource nodes for the modified znodes plugin that [[Balance of Power]] utilises. What this means is that each territory yields a given resource which is represented as a vanilla item, whose crafting recipes and functions have been changed.
